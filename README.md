# Cascaded Key Map

[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)

a concurrent map has cascaded keys like:
    put(value, key1, key2, ..., keyN)
    get(key1, key2, ..., keyN)
    remove(key1, key2, ..., keyN)
